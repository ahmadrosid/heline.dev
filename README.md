## Development
Run script build to run dev mode locally.
```bash
bash build.sh
```

Install required dependencies.
```bash
bash scripts/bootstrap.sh
```

Start solr
```bash
bash scripts/solr.sh start
```

Prepare solr index
```bash
bash scripts/solr.sh prepare
```
